# 🏨 StayNest – Hotel Booking Website

A modern and responsive **Hotel Booking Website** built using **HTML, CSS, and JavaScript**.
This project showcases a premium UI design with smooth layout, responsive cards, and clean structure suitable.

---

## 🚀 Features

* 🌐 Fully responsive design (Mobile, Tablet, Desktop)
* 🏨 Home page with hero section & search bar
* 📍 Destinations page with modern card layout
* 🛏 Rooms page with booking UI
* ℹ️ About Us page with clean layout and feature cards
* 🎨 Consistent UI/UX across all pages
* ⚡ Smooth hover effects and transitions
